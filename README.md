
# SuperMarket
#### My first project with C# fundamentals and Some of the principles of OOP  

## To run the program
* After Clone , Run "SuperMarket.sln"
* Copy all .txt files , Go to "SuperMarket \ bin \ Debug"
* Paste files , Run the program from Visual Studio
